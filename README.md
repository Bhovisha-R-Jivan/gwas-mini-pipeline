# GWAS Meta-Analysis Mini Pipeline

## Overview
This repository demonstrates a simplified workflow for performing a meta-analysis of GWAS data, inspired by my honours project on breast cancer in African populations.

## Workflow
1. **Data cleaning** : Uniform formatting of multiple datasets (Linux scripts)
2. **Meta-analysis** : Run Metal on cleaned datasets (Linux commands included)
3. **Visualisation** : Generate Manhattan plots using R

##Tools
Linux
METAL
R(ggplot2)
